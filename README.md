# Tetris
Tetris implementation using Scala and processing library

Keybindings:

move | player1 | player 2
------------ | ------------- |-------------
rotate left|V|K
rotate right|B|L
rotate right|W|up arrow
move down|S|down arrow
move left|A|left arrow
move right|D|right arrow
hold|N|J
hard drop|Space|Enter
